# Php Page Analyzer

## Description

This is a training project 3 for Hexlet PHP developer course.
It checks website pages how they suit for SEO.

## Demo

https://php-page-analyzer-hypu.onrender.com/

## Requirements

* php 8.2.0+
* composer

## Install

```sh
git clone https://github.com/eviktor/php-project-9.git
cd php-project-9
make install
```
